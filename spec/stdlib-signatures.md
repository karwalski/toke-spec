# toke Standard Library — Normative Signatures

**Status:** STUB — to be completed as part of Epic 1.3

This file is the normative interface for the toke standard library.
Changes here require coordinated updates to toke-stdlib and toke-benchmark.

## Modules

- `std.str` — string operations
- `std.http` — HTTP server and client
- `std.db` — database access
- `std.json` — JSON encode and decode
- `std.file` — file I/O
- `std.net` — TCP socket operations
